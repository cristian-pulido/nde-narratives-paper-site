# nde-narratives-paper-site

GitHub Pages website for the paper **"Quantifying representational mismatch between narratives and structured reports in near-death experiences."**

## Purpose

This repository provides a visual, concise project page for sharing the core scientific idea, key results, method overview, demo access, and citation details.

## Content source

The single source of scientific content is:

- `../nde-tone-valence-alignment/`

Figures in `assets/figures/` were selected from the paper figures and converted for web use.

## Local preview

From this directory:

```bash
python3 -m http.server 8000
```

Then open `http://127.0.0.1:8000`.

## GitHub Pages

This site is static (HTML/CSS/JS) and ready to publish from the repository root on the `main` branch.
